# Cockpit Transmission Shifter Unit

이 프로젝트는 STM32F103 (Blue Pill) 보드를 사용하여 자동차 변속기 위치를 읽고 CAN 버스를 통해 전송하는 장치입니다.

## CAN 통신 사양

*   **CAN ID**: `0x101` (Standard ID)
*   **DLC (Data Length Code)**: `2` bytes

### 데이터 페이로드

| 바이트 | 설명              | 값                                                                                                                              |
| :----- | :---------------- | :------------------------------------------------------------------------------------------------------------------------------ |
| 0      | 기어 상태 (숫자)  | `0`: N (중립)<br>`1`: 1단<br>`2`: 2단<br>`3`: 3단<br>`4`: 4단<br>`5`: 5단<br>`6`: 6단<br>`7`: R (후진) |
| 1      | 기어 상태 (ASCII) | `'N'`, `'1'`, `'2'`, `'3'`, `'4'`, `'5'`, `'6'`, `'R'`                                                                            |

## 핀맵 (Pin Map)

### 기어 입력

각 기어 위치는 GPIO 핀에 연결된 스위치로 감지됩니다. 핀은 내부 풀업 저항으로 설정되어 있으며, 스위치가 눌리면(기어가 선택되면) 해당 핀이 LOW 상태가 됩니다 (Active-low).

| 핀   | 기능     |
| :--- | :------- |
| PA0  | 1단      |
| PA1  | 2단      |
| PA2  | 3단      |
| PA3  | 4단      |
| PA4  | 5단      |
| PA5  | 6단      |
| PA6  | R (후진) |

### CAN 통신

| 핀   | 기능   |
| :--- | :----- |
| PA11 | CAN RX |
| PA12 | CAN TX |

### 상태 LED

| 핀   | 기능                      |
| :--- | :------------------------ |
| PC13 | 보드 내장 LED (Active-low) |

### USART 통신 (디버깅용)

| 핀   | 기능      |
| :--- | :-------- |
| PA9  | USART1 TX |
| PA10 | USART1 RX |
